## Applio Cloud Realtime :icon-cloud:
:::

Applio's Cloud versions (Google Colab, Kaggle, and Lightning.AI) support the **Realtime** tab. This allows users without a powerful local GPU to perform live voice conversion using cloud-hosted hardware.

***

### 1. Choose Your Cloud Provider
To use Applio on the cloud, follow the installation guide for your preferred platform. These guides will provide you with a URL (via Localtunnel, Ngrok, or Gradio) to access the Applio interface.

- [ :icon-book: **Applio Google Colab Guide** ](https://docs.aihub.gg/rvc/cloud/applio-colab/)
- [ :icon-book: **Applio Kaggle Guide** ](https://docs.aihub.gg/rvc/cloud/applio-kaggle/)
- [ :icon-book: **Applio Lightning.AI Guide** ](https://docs.aihub.gg/rvc/cloud/applio-lightning-ai/)

***
